# Fonts

This directory holds the Arabic and Latin fonts used for Quran verse overlays.

## Required fonts

| File                | Font                         | Source                                                  |
| ------------------- | ---------------------------- | ------------------------------------------------------- |
| `Amiri-Regular.ttf` | Amiri (Arabic naskh)         | [Google Fonts](https://fonts.google.com/specimen/Amiri) |
| `DejaVuSans.ttf`    | DejaVu Sans (Latin fallback) | [DejaVu Project](https://dejavu-fonts.github.io/)       |

## Auto-download

```bash
bash fonts/download_fonts.sh
```

Both fonts are open-source (OFL / Bitstream Vera license).
