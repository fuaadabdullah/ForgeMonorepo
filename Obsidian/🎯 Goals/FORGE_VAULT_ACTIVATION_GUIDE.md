### Common Issues & Solutions

**Plugins not loading:**

- Ensure plugins are installed from Obsidian Community Plugins
- Restart Obsidian after enabling plugins
- Check plugin settings in `.obsidian/plugins/*/data.json`

**Dataview queries not working:**

- Verify Dataview plugin is enabled
- Check that data files exist in expected locations
- Review query syntax in dashboard

**Templater functions failing:**

- Ensure Templater plugin is enabled
- Check that `forge-functions.js` exists in Scripts folder
- Verify function syntax and permissions
